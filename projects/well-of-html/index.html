Filter has two parameters, a collection and a function, and it returns a new array of filtered values. It loops over the collection and applies the function to each value in the collection. The function's parameters may be the value, the position and the collection. The function tests for true or false, and elements returning true are added to the filtered array.

Partition has two parameters, a collection and a function. The collection may be an array or an object. The function may have the parameters of value, position and collection. It tests for true or false. Partition returns an array of two arrays, the first containing values for which the test function returns true and the second containing values for which the test function returns false.

Every has two parameters, a collection and a test function. The collection may be an array or an object. The test function resolves to true or false. Every returns true if all tests return true and returns false if any test returns false.

Extend has at least two parameters which are objects (object1 and object2). It can have an unknown number of objects. If object 2 has different values for the same property key as object1, the value in object2 is copied to object1. If object2 has additional properties than object1, the new proprties are added to object1. If more than two objects are included in the extend call, the additional objects will be compared to object1 as above. The revised object1 is returned.
